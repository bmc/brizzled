Source for my [Brizzled][] blog.  Must be formatted with [Jekyll][].

NOTE: Courtesy of some monkeypatched hacks in `_plugins/page.rb`, [Liquid][]
tags can be escaped as follows:

    {\%   # yields {%
    \%}   # yields %}
    \{\{  # yields {{
    \}\}  # yields }}
